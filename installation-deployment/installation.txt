Installation
============

== Server Installation ==

1. Download the latest release from http://neo4j.org/download
   * select the appropriate version for your platform
2. Extract the contents of the archive
   * refer to the top-level extracted directory as +NEO4J-HOME+
3. Use the scripts in the +bin+ directory
4. Refer to the packaged information in the +doc+ directory for details

== Embedded Installation ==

To embed Neo4j in your Java application, use the Neo4j library jars in your build.
Either take the jar files from the +lib+ directory of the downloaded archive, or use the artifacts available from Maven Central.

.Maven dependency
--------------------------------------------
<project>
...
 <dependencies>
	<dependency>
   	<groupId>org.neo4j</groupId>
   	<artifactId>neo4j</artifactId>
   	<version>1.2.M04</version>
	</dependency>
	...
 </dependencies>
...
</project>

--------------------------------------------
