Installation, requirements and deployment scenarios 

Neo4j can be embedded into your application, run as a standalone server or deployed on several machines providing high availability.

Requirements

 o Java version 1.6 is required to run Neo4j. 
 o To deliver full ACID semantics it is depending on the operating system and underlying file system, a flush (fsync, fdatasync) must be persisted to the underlying storage. 
 o Recommended operating systems are Solaris and Linux, recommended filesystems include EXT3,EXT4 and ZFS.
